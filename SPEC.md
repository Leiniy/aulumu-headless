# aulumu-headless - SPEC

## 1. Concept

Headless Next.js frontend for Shopify store. Design reference: www.aulumu.com.

## 2. Design

Colors: primary=#1a1a2e, accent=#e94560
Font: Inter
Stack: Next.js 14, TypeScript, Tailwind CSS, Framer Motion

## 3. Components

- Header (sticky, scrolled state)
- TopBanner
- HeroSection
- IntroSection
- FeaturesGrid
- TechSection
- ProductShowcase
- WorkflowSection (carousel)
- TestimonialsSection (carousel)
- PressSection
- NewsletterSection
- TrustBadges
- Footer
- CookieConsent
- MobileMenu

## 4. Tech

- Next.js 14 App Router
- TypeScript
- Tailwind CSS
- Framer Motion animations
- Shopify Storefront API (optional)
- Netlify auto-deploy
